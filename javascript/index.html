Nesse bloco preciso que seja feita a validação do formulário, ou seja, os campos com marcação em * precisam ser validados antes do submit do form.
Obs.: a validação deve ser feita via JS, e não com o require padrão do html.

Nesse bloco preciso que seja feita a exibição de CPF ou CNPJ conforme selecionado,
caso seja escolhido CPF irá exibir o campo CPF, caso seja selecionado CNPJ, será exibido CNPJ.
Obs.: As validações de campo também devem ocorrer aqui.

Nesse bloco preciso que o nome clicado seja inserido tendo da tag com o id textClick.

BÔNUS

Nesse bloco temos um webservice, ViaCEP, que retorna um endereço quando é enviado um CEP para o serviço.
Itens a serem considerados:

  • A requisição deve ser feita via AJAX.
  • O consumo das informações deve ser via JSON.
  • As informações devem ser preenchidas dentro de cada input no form.
  • Apenas os inputs no form devem ser preenchidos.
  • O serviço a ser utilizado é ViaCEP.
  • A validação do campo CEP, pode ser feita com require do HTML.
